# @clipform/n8n-nodes-clipform

An [n8n](https://n8n.io) community node for [Clipform](https://clipform.io) - trigger a workflow whenever a form is completed. Works on n8n Cloud and self-hosted.

## Installation

In n8n: **Settings → Community nodes → Install**, then enter `@clipform/n8n-nodes-clipform`.

## Setup

1. Create a **Clipform API** credential with an API key from your Clipform dashboard (**Developer → API keys**).
2. Add the **Clipform Trigger** node and activate your workflow.

Full setup and usage: **[clipform.io/docs/guides/integrations/n8n](https://clipform.io/docs/guides/integrations/n8n)**.
